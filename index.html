Overview

A fixed-supply token with a visible burn schedule.

2SEC is a 1,000,000,000 supply token on Sui with a hard-coded burn vault and a phased burn rate. The contract never mints after initialization. All supply and burn logic is defined in code and verifiable on-chain.

Anyone can call the interact function. Each call advances the burn based on how many seconds have passed since the last interaction, using a higher rate for the first 200 days and a lower rate after that.

Burn mechanics

The burn vault is a shared object holding 85% of supply. A separate authority object holds the burn capability. The only exposed operation using that capability is burn inside interact. There is no way to mint new tokens after initialization.

Burn schedule

Phased burn: 15/sec then 2/sec.

From the moment of the first interaction, the contract tracks time in seconds and burns from the vault according to two phases:
